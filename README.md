# Deep Learning
## Individuell uppgift - Klassificera ansiktsuttryck med neurala nätverk
Målet är att använda ett neuralt nätverk för att bygga en modell som kan lära sig från bilddata, skilja mellan olika ansiktsuttryck och göra prediktioner på nya bilder.

Analysera data från filerna test och train, som innehåller bilder med olika ansiktsuttryck.
Resultatet presenteras i Jupyter Notebooken carina_karlsson_DL_individuelluppgift.ipynb

### Miljö

* Python: 3.13.7

## För att köra projektet
### Klona projetet
```
git clone https://github.com/Cym3ra/Deep_Learning_Individualassignment.git
```
## Skapa och aktivera virtuell miljö
```
python -m venv .venv
```
### Windows PowerShell
```
.venv\Scripts\Activate
```
### macOS/Linux
```
# source .venv/bin/activate
```
### För att installera nödvändiga paket:
```
pip install -r requirements.txt
```
